dual code


Let C be a linear code of block length n over the finite field 𝔽q. Then the set

C⟂:={d∈𝔽qn∣c⋅d=0⁢ for all ⁢c∈C}

is the dual code of C. Here, c⋅d denotes either the standard dot product or the Hermitian dot product.

This definition is reminiscent of orthogonal complements of http://planetmath.org/node/5398finite dimensional vector spaces over the real or complex numbers. Indeed, C⟂ is also a linear code and it is true that if k is the http://planetmath.org/node/5398dimension of C, then the of C⟂ is n-k. It is, however, not necessarily true that C∩C⟂={0}. For example, if C is the binary code of block length 2 http://planetmath.org/node/806spanned by the codeword (1,1) then (1,1)⋅(1,1)=0, that is, (1,1)∈C⟂. In fact, C equals C⟂ in this case. In general, if C=C⟂, C is called self-dual. Furthermore C is called self-orthogonal if C⊆C⟂.

Famous examples of self-dual codes are the extended binary Hamming code of block length 8 and the extended binary Golay code of block length 24.
